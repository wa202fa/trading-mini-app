# Trading Mini App (US + Saudi Tadawul)

Streamlit app for quick stock analysis with:
•⁠  ⁠Close price
•⁠  ⁠MA20 / MA50
•⁠  ⁠RSI14
•⁠  ⁠Chart + Excel export
•⁠  ⁠Supports US tickers (e.g., AAPL) and Saudi Tadawul tickers via .SR (e.g., 1120 -> 1120.SR)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
